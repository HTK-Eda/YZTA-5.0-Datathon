# YZTA 5.0 Datathon: Bilişsel Performans Tahmini
**Yarışma Amacı:** Katılımcı verilerinden yola çıkarak en doğru bilişsel skor tahminini yapmak.

## Uygulanan İşlemler
Analiz süreci şu aşamalardan oluşmaktadır:

1.  **Veri Ön İşleme:** Eksik verilerin doldurulması ve aykırı değer analizi.
2.  **Özellik Mühendisliği:** Tahmin başarısını artırmak için yeni değişkenlerin türetilmesi.
3.  **Model Eğitimi:** Regresyon algoritmaları (Linear, Ridge, Lasso veya Ensemble modeller) ile modelleme.
4.  **Değerlendirme:** Modelin hata payının (MSE, RMSE, R²) hesaplanması.


**Not:** Bu çalışma YZTA 5.0 Datathon süreci kapsamında geliştirilmiştir.
